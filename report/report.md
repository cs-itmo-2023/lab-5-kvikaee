## Использование Git Flow в проекте

1. Проверила, установлен ли Git FLOW:

```
sudo apt-get install git-flow
```

2. В корне репозитория во=ыполнила инициализацию Git Flow.

```
git flow init
```

3. Создала ветку для новой функциональности "task-management":

```
git flow feature start task-management
```

4. В файл task_manager.py добавила функционал управления задачами:
<img width="459" alt="Снимок экрана 2023-12-29 в 14 53 42" src="https://github.com/cs-itmo-2023/lab-5-kvikaee/assets/144216835/17b18867-bcc6-4794-bcf9-065fe1f966f2">


5. После завершения разработки функции объединилае ее с основной веткой:

```
git flow feature finish task-management

```

6. Переключилась на ветку develop и начала создание релиза:

```
git checkout develop
git flow release start v1.0.0
```

7. Внесла изменения, (обновила версию в файле version.txt):

```
echo "v1.0.0" > version.txt
git add version.txt
git commit -m "Обновлена версия для релиза v1.0.0"

```

8. Завершила релиз и объединила его main.
   
9. Завершила работу и отправила изменения на удаленный репозиторий.

```
git push origin develop
git push origin main

```
